# 🌌 Quantum Finance

> Gestão patrimonial de elite movida a Inteligência Artificial e precisão matemática absoluta.

O **Quantum Finance** é uma aplicação de finanças pessoais desenvolvida para oferecer análises profundas, previsões e categorização automática de gastos, operando sob uma arquitetura de alta performance e segurança militar.

---

## ✨ Funcionalidades Principais (Core Features)

* 🛡️ **Matemática Blindada (Zero Fluctuation):** O núcleo financeiro utiliza `decimal.js` e o "Padrão de Centavos" para garantir precisão absoluta e imunidade ao clássico erro de ponto flutuante do JavaScript (0.1 + 0.2 = 0.3).
* 🤖 **Motor IA Quântico (Gemini 1.5):** Categorização automatizada de extratos (CSV/OFX) e um Chatbot de mentoria financeira integrados via Google Cloud Functions.
* 🔒 **Conformidade LGPD & Privacidade:** Os dados enviados para a IA são rigorosamente anonimizados (remoção de CPFs, valores absolutos ocultados) e as auditorias (Logs) ignoram informações sensíveis.
* 📊 **Business Intelligence (BI):** Dashboards interativos e Análise de Pareto (Regra 80/20) para identificação visual de ralos financeiros.
* ⚡ **Arquitetura Resiliente:** Operações de escrita/exclusão em lote (Firestore `writeBatch`) garantem a integridade atómica dos dados (Tudo ou Nada), prevenindo dados fantasma em caso de falha de rede.
* 📱 **Mobile-First & Performance:** Fundo animado de partículas dinâmico com otimização extrema de bateria (*throttling* adaptativo e deteção de aba oculta via `visibilitychange`).

## 🛠️ Stack Tecnológico

**Frontend:**
* [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/) - Performance e renderização ultrarrápida.
* [Tailwind CSS](https://tailwindcss.com/) - Estilização utility-first (Design Glassmorphism).
* [Zod](https://zod.dev/) - Validação rigorosa de formulários e *schemas* de dados.
* [Recharts](https://recharts.org/) - Visualização gráfica de património e fluxo de caixa.
* [Decimal.js](https://mikemcl.github.io/decimal.js/) - Aritmética de precisão arbitrária.

**Backend & Infraestrutura:**
* [Firebase Authentication](https://firebase.google.com/docs/auth) - Login seguro (Google Auth).
* [Cloud Firestore](https://firebase.google.com/docs/firestore) - Base de dados NoSQL em tempo real com regras de segurança apertadas e limites de leitura.
* [Firebase Cloud Functions](https://firebase.google.com/docs/functions) - Servidor Node.js *Serverless* com proteção de Rate Limiting.

**Testes & Qualidade:**
* [Vitest](https://vitest.dev/) - Testes unitários do motor de conversão financeira e formatações de UI.
